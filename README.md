# 🚀 OptiBid (GoBidX)

A full-stack British Auction RFQ system where suppliers compete by lowering bids.
Features dynamic ranking (L1/L2/L3), automated time extension to prevent last-second bidding, and a strict forced close mechanism for fair auctions.

## ⚙️ Tech Stack

* FastAPI (Backend)
* PostgreSQL + SQLAlchemy
* React (Vite)
